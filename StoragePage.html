Computer Storage

Hard Disk Drive

The hard drive is where information and data get stored long term. This includes any file you save on your computer, and all the code required for the operating system to boot up and function properly. Each hard drive has a specific amount of storage space. Hard drives use a spinning magnetic disk(platter) in which information is stored in small tracks located on the disk. The data is written inside the tracks by an extended actuator head.

Solid State Drive

The most noticeable difference between a HDD and SSD is that SSDs require no internal moving parts (hence the name “Solid-State”). SSD’s serve the same purpose as traditional hard drive disks but come in a much smaller and efficient form factor.
